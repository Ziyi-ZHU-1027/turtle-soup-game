我将为你制定一套全新的**“暗黑侦探卷宗”（Cryptic Elegance）**视觉与交互升级方案，并附带高保真 Markdown 视觉框架图。

### 第一阶段：视觉语言与设计系统定位 (Design System)

我们要把当前的“简单深色主题”升级为**“悬疑侦探的数字工作台”**，融合“复古纸质卷宗”与“现代赛博数字”的风格。

#### 1. 色彩规范 (Color Palette)
用色彩直接传递游戏状态，减少玩家阅读成本：
*   **深邃背景 (Midnight Desk)**: `#0D0F12` (比纯黑更透气，带有极暗的蓝灰倾向)
*   **面板材质 (Frosted Glass)**: `rgba(26, 28, 35, 0.6)` + 16px 背景模糊 (Backdrop Blur)
*   **主品牌色/接近答案 (Amber Glow)**: `#D4AF37` (复古金，用于高亮、接近答案时的脉冲光效)
*   **肯定反馈/正确 (Neon Mint)**: `#2A9D8F` (是/部分正确)
*   **否定反馈/错误 (Crimson Fade)**: `#E63946` (不是/无关)
*   **字体层级**:
    *   一级（纯白）：`#FFFFFF` (汤面描述)
    *   二级（灰白）：`#E0E0E0` (正常聊天)
    *   三级（幽灰）：`#808596` (时间戳、辅助信息)

#### 2. 字体排版 (Typography)
*   **汤面/卷宗区**: 采用**衬线体**（如 Noto Serif SC / 思源宋体），营造纸质书和悬疑小说的氛围。
*   **对话/UI区**: 采用**无衬线体**（如 Inter / 思源黑体），保证长时间阅读和数字交互的清晰度。

### 第二阶段：核心交互体验升级 (Interaction Upgrades)

#### 1. 状态可视化（核心亮点）
将 AI 的六种回复类型做**视觉降维**，让玩家一眼看出自己的方向对不对：
*   ✅ **是**：气泡带微弱绿色边框，出现清脆的“滴”声（如有音效）。
*   ❌ **不是**：气泡文字带有轻微的划线感或红色渐变，表示“此路不通”。
*   🔥 **接近答案**：气泡**高亮描边**，触发金色呼吸灯动效，激发玩家的荷尔蒙。
*   💡 **提示盲区**：不使用普通气泡，而是以“拍立得相片”或“便签纸”的形态插入聊天流中。

#### 2. 汤面卡片“卷宗化”
目前的“汤面卡片”容易占用空间。优化为：**顶部悬浮的折叠卷宗**。滚动对话时，它会缩为只有一句话的 Header，点击即可展开查看详情。

#### 3. “输入流”的防身术
由于玩家很容易陷入“死胡同”，在输入框上方增加**动态预判引导**。当玩家卡住时，输入框里的 Placeholder 可以从“输入你的问题...”变成“不妨问问看：受害者的职业是？”

### 第四阶段：给前端 / 开发团队的落地执行建议

为了将以上设计完美落地，建议在前端技术栈中引入以下工具：

#### 1. CSS 框架建议：引入 Tailwind CSS + VueUse
*   **抛弃纯手写 CSS**：当前 MVP 的内联样式和独立 CSS 难以维护。引入 **Tailwind CSS**，可以非常方便地实现暗黑模式（`dark:bg-slate-900`）和毛玻璃效果（`backdrop-blur-md bg-white/10`）。
*   **VueUse**：使用 `@vueuse/core` 中的 `useScroll` 自动处理消息滚动到底部，`useWindowSize` 处理响应式断点。

#### 2. 动画库建议：Framer Motion for Vue (@vueuse/motion 或 AutoAnimate)
*   **AI 流式打字机效果**：除了光标闪烁，可以加上文本淡入的效果。
*   **状态切换动效**：当回复是“接近答案”时，使用 CSS Keyframes 增加一个 `pulse-gold` 的发光外发光（box-shadow）动画。
*   **FormKit / AutoAnimate**：只要加一行指令 `v-auto-animate`，聊天记录的增加、卷宗的折叠就会自动拥有丝滑的过渡动画，极大提升高级感。

#### 3. 组件级优化清单 (Component Refactor Checklist)
*   [ ] **`MessageBubble.vue` 拆分**：
    *   接收 `type` Prop (`success`, `error`, `close`, `hint`, `normal`)。
    *   根据 `type` 渲染不同的边框、背景色和前缀 Icon。
*   [ ] **`PuzzleCard.vue` 优化**：
    *   增加“折叠/展开”状态。
    *   标题使用衬线体，增加纸张纹理背景（可使用极低透明度的 noise.png）。
*   [ ] **`CelebrationEffects.vue` 增强**：
    *   除了前端用 `canvas-confetti` 库放烟花，当猜中时，**整个界面的背景**可以从暗黑瞬间提亮，象征“真相大白（破晓）”。

通过这套**“暗黑侦探”**的包装，玩家的每一次提问都不再是干巴巴的文字交互，而是一次情绪起伏的破案体验。这样的设计不仅解决了 MVP 视觉简陋的问题，还能极大提升用户的留存率和分享欲。
